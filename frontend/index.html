Wheather from Photos

Simple model based on Resnet34, trained on results from Google Image search, using FastAI (on top of Pytorch) with to categorise images as either Sunny or Raining.

The backend is a Python Flask API with frontend CSS framework Bulma

[Could add JS framework e.g. Svelte/Vue/React at some point, but it's overkill at this stage].
